# Model Comparison Day 75 Decision Cards Summary

Day 75 renders the three API-provided Decision Cards in the Model Comparison
view: observed metric leader, selected integration candidate and evidence
boundary. Each card shows its metric, reasons and limitation through semantic
article, heading, status and list elements.

The frontend view model preserves the API identifier, model reference, metric
value, direction, reasons and limitation. It maps only machine status values to
display labels; it does not rank candidates or run the selection policy.

Cross-layer smoke validation and Week 7 closure remain assigned to Days 76 and
77 and are not claimed by this summary.
