# Model Comparison Report Template Evaluation

| Template | Human review | Machine validation | Decision context | Selected |
|---|---|---|---|---|
| Narrative-first | strong | weak | strong | no |
| Table-only | medium | medium | weak | no |
| Evidence-first composite | strong | strong | strong | yes |

The selected template keeps one versioned JSON report and generates Markdown
from the same evidence. This is a documentation experiment, not model evidence.
