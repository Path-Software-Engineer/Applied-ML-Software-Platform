# TEC Lab — Model Error Analysis

## Question

What does the sign of a residual reveal that absolute error cannot?

## Controlled experiment

The lab creates two predictions with the same absolute error but opposite
residual signs. It demonstrates under- and over-prediction terminology without
using official Sprint 2 rows.

## Boundary

- The lab is not imported by production code.
- Residual direction is descriptive, not causal.
- Controlled values do not affect formal model selection.
