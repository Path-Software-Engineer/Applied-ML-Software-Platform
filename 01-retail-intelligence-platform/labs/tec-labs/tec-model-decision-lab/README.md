# TEC Lab — Model Decision

## Question

Can the lowest metric and the preferred engineering candidate be different?

## Controlled experiment

Two hypothetical learned candidates differ by only 0.10 MAE units. The lab
applies the same practical-equivalence and complexity policy as production
decision code.

## Boundary

- Controlled candidates are not official model results.
- The lab is not imported by production code.
- Selection means next learning integration, not production deployment.
