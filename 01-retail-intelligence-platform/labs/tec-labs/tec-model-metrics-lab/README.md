# TEC Lab — Model Metrics

## Question

Why should MAE and RMSE be read together instead of treated as interchangeable?

## Controlled experiment

Two hypothetical predictors have different error shapes. One spreads small
errors across observations; the other contains a single large miss. The lab
uses the production metric formulas but not the official retail evidence.

## Boundary

- The lab is not imported by production code.
- Controlled values are not candidate results.
- The experiment explains metric sensitivity and does not select a model.
