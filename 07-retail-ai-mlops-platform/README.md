# 07-retail-ai-mlops-platform

## 🧠 Descripción

Proyecto estrella del Plan 1 — AI / Machine Learning Engineering.

Plataforma retail de IA que integra predicción de demanda, forecasting, monitoreo MLOps, drift, retraining, RAG, costos, latencia, dashboard y frontend serio.

Este proyecto une las habilidades trabajadas en los proyectos 01 al 06.

---

## 🎯 Objetivo

Construir una plataforma completa que integre servicios de IA, MLOps, RAG, monitoreo, costos, latencia y visualización de negocio.
