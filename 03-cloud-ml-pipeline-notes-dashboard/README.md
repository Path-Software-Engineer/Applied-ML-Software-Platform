# 03-cloud-ml-pipeline-notes-dashboard

## 🧠 Descripción

Dashboard ligero de notas y arquitectura para visualizar un flujo ML local y su equivalente cloud.

Este proyecto no busca desplegar un sistema cloud real desde el inicio. Su objetivo es convertir conceptos cloud en un mapa visual entendible, comparando GCP, AWS y Azure desde la perspectiva de un pipeline ML.

---

## 🎯 Objetivo

Crear una herramienta ligera para documentar y visualizar un flujo ML cloud:

```txt
Pipeline local → mapa GCP → equivalencias AWS/Azure → notas de arquitectura → tarjetas visuales → README visual / dashboard
```

---

## 👤 Usuario objetivo

* Estudiante de AI Engineering.
* Persona que quiere entender cloud aplicado a ML.
* Reclutador técnico que quiere ver criterio de arquitectura.
* Equipo técnico que necesita una explicación simple de equivalencias cloud.
