# 07-ai-portfolio-platform-lite

## 🧠 Descripción

Portal ligero para agrupar proyectos, aprendizajes, demos y evidencias del Plan 1 de Building Projects.

Este proyecto funciona como cierre visual del primer tramo: reúne microproductos, links, capturas, aprendizajes y estado de avance en una vista simple y presentable.

No busca ser una plataforma gigante. Busca ser un portafolio claro, ordenado y fácil de mostrar.

---

## 🎯 Objetivo

Crear un portal ligero que agrupe proyectos, aprendizajes, demos y evidencias:

```txt
Proyectos → tarjetas → capturas → links → aprendizajes → roadmap → portal ligero
```
